WEB BROWSER:- A web browser takes you anywhere on the internet. It retrieves information from other parts of the web and displays it on your desktop or mobile device. WEB PAGE:- A document which can be displayed in a web browser such as Firefox, Google Chrome, Opera, Microsoft Edge, or Apple Safari. These are also often called just "pages." WEBSITE:- A collection of web pages which are grouped together and usually connected together in various ways. Often called a "web site" or a "site." WEB SERVER:-A computer that hosts a website on the Internet. HTML:- HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page's words and images. CSS:- Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.
